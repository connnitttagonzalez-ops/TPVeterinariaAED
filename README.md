## Escenario asignado: Gestión de veterinaria
```
El sistema deberá administrar turnos y atención de mascotas dentro de una veterinaria. La solución podrá incluir registro de mascotas, turnos, atención médica y control de servicios realizados.

También podrá incorporar estadísticas básicas relacionadas con tipos de consultas o servicios más frecuentes.
```
## Instrucciones de ejecución:
```
Al iniciar aparece un menú con las opciones.
Primero registrás a la mascota con la opción 1, que te va a pedir sus datos y al final te devuelve un ID con el que se irá operando a continuación.
Con la opción 2 solicitás un turno usando el ID de la mascota, elegís fecha y hora y el programa te da un ID de turno.
Con la opción 3 pedís un servicio (baño, peluquería, venta de productos o atención médica) ingresando el ID del turno.
Con la opción 4 podés ver todos los servicios realizados junto con las estadísticas, y con la opción 5 salís del programa.
```
